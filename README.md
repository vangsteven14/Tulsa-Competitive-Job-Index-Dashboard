# Tulsa-Competitive-Job-Index-Dashboard

Greater Tulsa lacks baseline data on whether job creation efforts are producing jobs meeting quality standards (e.g. fair pay, benefits, stability, growth opportunities).<br>
The Tulsa Competitive Job Index dashboard is a project which addresses underlying concerns in Tulsa County’s workforce. It is a product which brings awareness to Greater Tulsa’s current job quality and status. <br>

This project covers Six Core Dimensions which includes: <br>
  ● <ins>Living Wages:</ins> Ensures compensation meets the baseline for economic stability.<br>
  ● <ins>Health and Wellness:</ins> Provides a safety net for physical well-being and recovery from occupation injury.<br>
  ● <ins>Professional Development:</ins> Offers clear pathways for skill acquisition and career advancement.<br>
  ● <ins>Competitive Wage:</ins> Positions pay at the top of the regional market for the specific role.<br>
  ● <ins>Flexibilty:</ins> Grants workers control over their time through predicatable scheduling and flexibility. <br>
  ● <ins>Family Planning and Retirement:</ins> Delivers long-term security through structured savings and deferred compensation.<br>
